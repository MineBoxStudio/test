Integration Guide

To integrate the splash screen into your project, follow these steps:

1. Include the script

<script src="splash-screen.js"></script>

2. Initialize the splash screen

// Basic usage
const splash = new SplashScreen();

// With custom options
const splash = new SplashScreen({
    backgroundColor: '#FF5722',
    logoSrc: 'path/to/logo.png',
    logoWidth: 150,
    logoHeight: 150,
    text: 'Loading...',
    textColor: '#FFFFFF',
    duration: 3000,
    showProgressBar: true,
    progressBarColor: '#FFFFFF',
    onHide: () => {
        console.log('Splash screen hidden');
    }
});

3. Control methods

// Hide the splash screen manually
splash.hide();

// Show the splash screen again
splash.show();

// Update options
splash.updateOptions({
    backgroundColor: '#4CAF50',
    text: 'Almost there...'
});

4. Available options

// All available options with default values
{
    containerId: 'splash-screen-container',  // ID for the container
    backgroundColor: '#ffffff',              // Background color
    logoSrc: null,                           // URL to logo image
    logoWidth: 120,                          // Logo width in pixels
    logoHeight: 120,                         // Logo height in pixels
    logoAlt: 'Logo',                         // Alt text for logo
    text: null,                              // Text to display
    textColor: '#333333',                    // Text color
    duration: 2500,                          // Duration in milliseconds
    fadeOutDuration: 500,                    // Fade out animation duration
    showProgressBar: false,                  // Show progress bar
    progressBarColor: '#4CAF50',             // Progress bar color
    showSpinner: false,                      // Show loading spinner
    spinnerColor: '#333333',                 // Spinner color
    onHide: null,                            // Callback after hiding
    zIndex: 9999                             // Z-index for the splash screen
}
